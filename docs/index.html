📋 How Manual Signals Work
Manual signals are stored in Google Sheets so they appear on every device — phone, desktop, any browser.
After adding a row, click ↻ Refresh Data below or use ▶ Run Scanner in the header to trigger a workflow run and pull the new record into the dashboard.

Opens a Google Form pre-configured for the Manual Signals tab. Works from any device. Responses go directly to the Sheet.
